# 个人网站构建（Hugo + github）

总体思路：自己写 Markdown 文件（存储在 content 目录），Hugo设置模版并把我们的 Markdown 转换为静态文件中（存储在 public 目录），最后提交到 github 仓库并托管。

### 1.Hugo: markdown 转换为静态网页

> 一个静态网站生成器，用 Go 语言编写，自己写好markdown，帮我转换为html静态网页。

#### 基础命令

```
# 安装 Hugo
brew install hugo  # macOS

# 创建新项目
hugo new site mysite

# 新建一篇文章
hugo new posts/my-first-post.md

# 本地预览
hugo server -D
```

#### 配置文件: hugo.toml常见的配置项

```
# ── 基础 ──────────────────────────────
baseURL = 'https://yoursite.com/'
title = '我的博客'
theme = 'your-theme'
languageCode = 'zh-cn'
defaultContentLanguage = 'zh'

# ── 内容处理 ──────────────────────────
hasCJKLanguage = true      # 中文字数统计
summaryLength = 60         # 摘要长度
paginate = 10              # 每页文章数
enableEmoji = true         # 支持 emoji

# ── 构建 ──────────────────────────────
cleanDestinationDir = true # 构建前清空 public/
buildDrafts = false        # 不构建草稿
buildFuture = false        # 不构建未来日期文章

# ── URL ───────────────────────────────
relativeURLs = false
canonifyURLs = false

# ── Markdown 渲染 ──────────────────────
[markup.goldmark.renderer]
  unsafe = true            # 允许 HTML 标签

[markup.highlight]
  style = 'github'         # 代码高亮风格
  lineNos = true           # 显示行号

# ── 主题自定义参数 ─────────────────────
[params]
  author = "你的名字"
  math = true
  show_read_time = true

# ── 导航菜单 ──────────────────────────
[[menu.main]]
  name = "首页"
  url = "/"
  weight = 1

# ── 输出格式（RSS/搜索索引等）──────────
[outputs]
  home = ['HTML', 'RSS', 'JSON']
```
